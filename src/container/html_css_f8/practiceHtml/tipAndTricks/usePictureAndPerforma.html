tối ưu sử dụng hình ảnh trên trang web
  1. Sử dụng hình ảnh phù hợp
    • Hình học, vecter... SVG
    • bitmap transparent: PNG, WEBP
    • bitmap không transparent: JPEG, WEBP
  2. Nén hình ảnh giảm dung lượng
    • https://tinypng.com/
  3. Sử dụng kích thước ảnh phù hợp
    sử dụng ít hình ảnh
    • ảnh ban đầu(original): 4000px X 4000px,
      thực tế thiết kế: 800px * 800px (css),
      vậy để phù hợp hiển thị trên nhiều màn khác nhau nên Resize:1600px * 1600px (2x)
      dáp ứng tốt cho cả màn hình PPI/DPI cao (retina)
      PPI (Pixel per inch (cho màn hình)) DPI (dots per inch(màn hình và in ấn)) (đo mật độ điểm ảnh)
    4. Loại màn hình

    1. màn hình thường ( DPI Thấp -1:1) tức là css 1 picxel tương ứng 1 picexl vật lý ( nếu thiết kế 800 *800 đưa resize về 800 *800 là ok)
    2. màn hình retina (DPI cao - 2:1) (sủ dụng 800 *800 )(nó sẽ dùng 1600 *1600 pixel vật lý để thể hiện)
Trang web sử dụng nhiều hình ảnh ( nếu dùng các cách tối ưu trên ) Thiết bị:
  • Mobile: 1600 * 1600 tốn bằng thông, chờ lâu hơn => giảm trải nghiệm
  • Tablet: 1600 * 1600
  • Desktop:1600 * 1600 ok
    • Bài toán:
  • Mobile: 1000px * 50px
  • Tablet: 2000px * 100px
  • Desktop: 4000px * 200px
